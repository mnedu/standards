INSTALLATION:
 - Drop parent directory ('Rename to "MState"') into the phpcs/CodeSniffer/Standards directory.


USAGE:

 - $> phpcs --standard=MState /path/to/directory/*
 - See further phpcs docs here: http://pear.php.net/package/PHP_CodeSniffer/docs


MISC:

 - This standards set is a copy of the Pear one, with our dislikes stripped out.
 - e.g.: Shamelessly ripped off and tweaked.

- Works for PHPCS 1.3
